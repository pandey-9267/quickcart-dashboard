# QuickCart Dashboard

QuickCart Dashboard is a simple e-commerce-style product browsing interface built with HTML, CSS, and vanilla JavaScript. The app loads product data from the Fake Store API and lets users search products, filter by category, sort by price, and increase a cart counter.

## Live Demo

- Visit the deployed app at: https://quick-cart-dash-board.vercel.app/

## Features

- Fetches product data from a public API
- Displays products in a clean card-based layout
- Supports live search by product title
- Filters products by category
- Sorts products by price from low to high or high to low
- Updates a cart count when a product is added

## Project Structure

- index.html: Main structure of the dashboard UI
- style.css: Styling for the page layout and product cards
- script.js: Logic for fetching products, filtering, sorting, and cart updates


## How to Run

1. Open the project folder in a browser.
2. Launch index.html directly, or use a live server for a smoother experience.
3. The app will load products automatically from the API.

## Notes

- The app requires an internet connection because it fetches data from the Fake Store API.
- If the API request fails, the page will display an error message.
